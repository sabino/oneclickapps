One Click Apps

Custom CapRover one-click apps published as a third-party repository. Browse what is available here, then add this repository inside CapRover to install the apps directly from the dashboard.

Add this repository in CapRover

  1. Open Apps in CapRover.
  2. Open One Click Apps.
  3. Scroll down to 3rd party repositories.
  4. Paste https://oneclickapps.sabino.pro and click Connect New Repository.
